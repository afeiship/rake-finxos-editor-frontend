# rake-finxos-editor-frontend
> Rake for editor-frontend.

## install:
```bash
npm i -S afeiship/rake-finxos-editor-frontend --registry=https://registry.npm.taobao.org
```

## usage:
~~~
rake app:build[env]  # Build app
rake app:pack        # Package dist file
~~~
